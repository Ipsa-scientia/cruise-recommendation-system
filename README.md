# Cruise Recommendation System – Система рекомендации круизов
Cruise Recommendation System – это удобное и умное веб-приложение для любителей круизов. Оно помогает экономить время, предлагая персонализированные рекомендации круизов на основе предпочтений пользователя, понравившихся и непонравившихся круизов, ценового диапазона, длительности и других характеристик.
Приложение создано с использованием Flask и использует контентно-ориентированную фильтрацию и метрики схожести для подбора наиболее подходящих круизов.
## Front-end
Разработан с использованием HTML и CSS.
Интерактивный интерфейс для выбора понравившихся и непонравившихся круизов, применения фильтров и просмотра рекомендаций.
## Back-end
Разработан на Flask.
Использует Pandas и NumPy для обработки данных.
Реализует косинусное сходство, манхэттенское и евклидово расстояние для генерации рекомендаций.
## Как запустить?
### Шаги:
Клонируйте репозиторий на свой компьютер:
```git clone https://github.com/<ваш-username>/cruise-recommendation-system.git```
```cd cruise-recommendation-system```
Установите зависимости:
```pip install -r requirements.txt```
Создайте файл .env в корне проекта:
```SECRET_KEY=ваш_секретный_ключ```
```FLASK_ENV=development```
```FLASK_DEBUG=True```
Поместите ваш CSV-файл с данными круизов в папку data/.
Убедитесь, что путь в config.py указан правильно:
```CSV_FILE_PATH = os.path.join(os.path.dirname(__file__), 'data', 'cruises_data.csv')```
Запустите сервер:
```python app.py```
Откройте браузер:
http://127.0.0.1:5001
## Используемые алгоритмы
### Контентно-ориентированная фильтрация
Система учитывает характеристики самих круизов: сезонность, цену, длительность, теги и наличие русскоязычных групп.
Рекомендует круизы, похожие на понравившиеся пользователю, и избегает тех, которые пользователь отметил как непонравившиеся.
### Метрики схожести
Косинусное сходство – измеряет угол между векторами признаков (например, сезонность).
Манхэттенское расстояние – измеряет разницу по абсолютным значениям (например, цена).
Евклидово расстояние – измеряет прямое расстояние в многомерном пространстве (например, теги).
### Оценка схожести
Similarity score показывает, насколько круиз соответствует предпочтениям пользователя:
Значение от 0 до 1.
Чем выше оценка, тем больше сходство с понравившимися круизами и меньше сходство с непонравившимися.
Оценки по всем признакам (сезон, цена, теги) суммируются с весами, чтобы получить топ-10 рекомендаций.
